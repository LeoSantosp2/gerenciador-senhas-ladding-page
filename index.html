Gerenciador de Senhas

Sobre o App

Você é aquela pessoa que sempre se esquece do e-mail ou senha de uma conta? Este app é para você! O Gerenciador de Senhas é um app com o intuito do usuário armazenar as credenciais de acesso. Pode ser armazenado qualquer dado de acesso. Instagram, Facebook, e-mails, bancos, etc… Utilizando o Gerenciador de Senhas, você não vai mais precisar criar uma nova conta porque se esqueceu do e-mail ou da senha.
