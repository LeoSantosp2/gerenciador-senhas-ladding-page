Sobre o Gerenciador de Senhas

Você é aquela pessoa que sempre se esquece do e-mail ou senha de alguma de suas contas? Este app é para você! O Gerenciador de Senhas, visa fornecer aos usuários uma forma de gerenciar os dados de acesso de diversas contas que temos pela internet a fora. Nele você pode salvar os dados de acesso de qualquer conta como redes sociais, e-mails, bancos e etc... Faça o download gratuito agora.
